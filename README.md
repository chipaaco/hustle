# hustle

Ultra complex productivity (CLI) tool, you'll waste more time on this than doing real work.

```
:'=
hustle

track tasks to earn leisure. 3 useful = 1 reward.

commands
1: useful (+1 aura)
0: leisure (-3 aura)
$: aura points
?: help
q: quit

example

:'=
hustle
* 1
['1', ' ', ' ', ' ', ' ', ' ']

:'=
hustle
* 1
['1', '1', ' ', ' ', ' ', ' ']

:'=
hustle
* 1
['1', '1', '1', ' ', ' ', ' ']

:'=
hustle
* 0
['1', '1', '1', ' ', ' ', ' ']
[' ', ' ', ' ', ' ', ' ', ' ']
:)
```

install
